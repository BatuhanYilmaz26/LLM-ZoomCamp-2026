# LLM Zoomcamp 2026 - Personal Learning Repository

This repository contains my personal journey, projects, homework, and code exercises for the **[LLM Zoomcamp 2026](https://github.com/DataTalksClub/llm-zoomcamp)**, a free hands-on course about real-life applications of Large Language Models (LLMs) organized by **[DataTalks.Club](https://datatalks.club/)**.

Through this course, I will be applying my knowledge to implement practical, production-ready AI systems that answer questions using customized knowledge bases. Key focus areas include Retrieval-Augmented Generation (RAG), vector search, embeddings, AI agents, evaluation, monitoring, hybrid search, and pipeline orchestration.

---

## 📚 Course Syllabus

Here is the outline of the modules covered in this course:

### Module 1: Agentic RAG

* Build a RAG pipeline with keyword search
* Make it agentic with function calling
* *Status: Basic RAG pipeline setup, working with `minsearch` and `dotenv`*

### Module 2: Vector Search

* Semantic search with embeddings
* `minsearch`, `sqlitesearch`, and `PGVector`

### Module 3: Orchestration

* AI orchestration with Kestra

### Workshop: Data Ingestion

* Build `dlt` pipelines to ingest and analyze LLM traces
* Filesystem and REST API sources, DuckDB, and `marimo` dashboards

### Module 4: Evaluation

* Measure retrieval and answer quality
* Offline and online evaluation

### Module 5: Monitoring

* Monitor user feedback and system health
* Live dashboards

### Module 6: Best Practices

* LangChain
* Hybrid search: combine vector and keyword search
* Rerank results for higher precision

### Module 7: End-to-End Project

* A complete project example: a fitness assistant built with LLMs

### Capstone Project

* Ship a complete end-to-end project of choice from scratch
